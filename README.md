### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/xokirBios/PepeSecurity/
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
3. Enter your TOKEN in `/botconfig/config.json`
  `"token": "TOKEN"`
